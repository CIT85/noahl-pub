Osu! Beatmap Recommender

What is it?

A site Recommends an Osu! Beatmap based on a players top play and preferred mapping style.

How does it work?

Obtaining Beatmaps

Osu! has separate categories for it's beatmaps, but the category that is most important is ranked. I actually ended up making a separate tool in Python to sift through every single ranked beatmap in the game and then save them to their specified JSON array. Once the data was obtained and sorted I was able to create a function in JavaScript that gets every map from a players preferred skillset, or mapping style. Then, it shuffles the array and returns a random beatmap as a recommendation.

Processing Data

When a player submits their data, a lot of things need to be processed. For starters, a beatmaps performance value is not saved, so the performance value that a person submits gets turned into a 'recommended star rating'. This recommended star rating gets compared to a beatmaps star rating inside of the JSON list. The star rating inside of the list gets multiplied depending on modifiers selected as well, Hidden is -0.25 Stars, Hardrock is *1.075 Stars, and Double Time is *1.4 Stars. After a beatmap is selected, it then prints the data to a table where a person can see the beatmap, its information, and delete it from the table.

Future for the Project.

Future plans for this project, along with any other thoughts and comments I have.

Overall Comments

For the most part the tool is done, you can give it your stats and it will give you a map. There are some things that I would change though if I were to remake the tool. For starters, it is currently limited to mapsets from 2007-2024, this is because I made the tool in 2024 so the JSON file containing all the ranked beatmaps is doesn't contain any maps made in the past 12 months. Ideally, I would use Osu!'s API in the site itself, to make a call for a random ranked map and do a check if it meets the criterias the player has listed.

With that, the only other comments I have are minor things. For example, a person should be able to set the amount of maps they want to generate, right now it's one at a time. Furthermore, it should save the stats you had just used to allow for generating multiple recommendations more easily. The rest of the site could also be filled with more stuff, possibly more options for loved and graveyarded maps as well. Those will likely be things I try out in the future though since I'm fairly satisifed with this project as is.
